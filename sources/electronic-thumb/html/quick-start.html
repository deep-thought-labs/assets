Electronic Thumb — Quick start

Get the widget on your page in a few steps. Below are the essential steps to configure and embed the widget, a full HTML template you can copy, optional CSS, a preview, and the state the widget exposes. For full customization options and a complete picture of how the widget works, see Architecture and Styling.

This page and the widget preview work when served over HTTP (not opened as a file). To try the widget locally, Node.js is required. Run the command below inside the folder where your HTML file is located (e.g. the folder where you saved the example):

npx serve .

In production, your host serves the site; no extra step.

Essential steps

1. Create or load the config before the widget script.
Set window.ElectronicThumb so it exists when the script runs.

<script>
  window.ElectronicThumb = {
    connectButtonLabel: "Connect",
    onStateUpdate: function (state) { console.log(state); }
  };
</script>

2. Put the container div where the widget should appear.
Use the electronic-thumb-widget attribute on a single div.

<div id="my-electronic-thumb" electronic-thumb-widget></div>

3. Load the widget script after the div.
Include the script once the container exists in the DOM.

<script src="https://assets.infinitedrive.xyz/widgets/electronic-thumb/electronic-thumb.js"></script>

Full HTML template. Copy and adjust the script URL to your environment.
